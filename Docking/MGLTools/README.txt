===============================================================================

		      MGLTOOLS version 1.5.7
		      BINARY DISTRIBUTION

===============================================================================

Thank you for downloading MGLTools version 1.5.7

This distribution contains a collection of pure Python packages (including Pmv, 
AutoDockTools,  DejaVu, MolKit, ViewerFramework, Vision, NetworkEditor, etc.), 
various platform dependent packages (including mslib, opengltk, bhtree, gle, 
stride, sff, cAutoDock, UTpackages, etc.), and the PMV, ADT and VISION applications 
built on top of these software packages.



**************************************************************************
